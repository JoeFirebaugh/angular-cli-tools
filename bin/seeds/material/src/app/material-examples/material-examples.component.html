How to add Material Design elements to you project

NOTE: As of this release Angular Material is in a (very) alpha state (Alpha.8)....meaning, half the stuff doesn't work.

Angular Material packages such as Material Button (@angular2-material/button) are installed easily by doing:

npm install --save @angular2-material/button

After installation you an import them into a module in your ng2 project:

{{
    "import { MdButtonModule } from '@angular2-material/button';\n"+
    "import { MdCardModule } from '@angular2-material/card';\n"+
    "// other imports \n"+
    "@NgModule({ \n"+
    "  imports: [MdButtonModule.forRoot(), MdCardModule.forRoot()],\n"+
    "  ... \n"+
    "})"}}

Then just include it in your component:

    <button md-raised-button>RAISED</button>
